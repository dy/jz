jz examples

Valid js is valid jz. Each demo runs the very same source two ways — imported as JavaScript, and compiled ahead-of-time to WebAssembly. Flip the js ⇄ jz switch in any demo's corner to watch the frame-time gap, live. Black is space, white is signal.
