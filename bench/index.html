06Methodology

  • Protocol — each case prints median_us · checksum · samples · stages · runs: the median of 21 runs after warmup; samples is the per-run workload count (matrices multiplied, bytes hashed).
  • Parity — ok: checksum matches the cross-target majority. DIFF: shown red, excluded from geomeans, never hidden. fma: Go's arm64 a*b+c fusion — same algorithm, last-bit rounding.
  • Subsets — geomeans cover what each target runs; the case count is printed on every row. json native rows hand-parse a fixed schema, not runtime JSON.parse. NumPy rows are C library calls — context, not contenders; excluded from the native-speed stat.
  • Sources — JS rows share the byte-identical file; AS and native rows are ports of the same algorithm.
  • Your machine — a tab is a noisy lab: ratios replicate, absolute µs vary. Live results never replace the reference.
